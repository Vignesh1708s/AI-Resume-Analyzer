# AI Resume Analyzer
## Project Description

AI Resume analyzer is a web application tahr analyzes resumes and provides:
--> Resume Score
--> Detected Skills
--> Missing Skills
--> Job Recommendation
--> Resume Feedback
--> Name,Email, and Phone Extraction

## Technologies Used:

--> Frontend
        -- HTML
        -- CSS
        -- JavaScript


--> Backend
    -- Python
    -- Flask

## Libraries:
--> PyPDF2
--> Flask-CORS

## Features:

-- Upload Resume PDF
-- Extract Resume Content
-- Detect Technical Skills
-- Calculate Resume Score
-- Suggest Missing Skills
-- Recommend Suitable Job
-- Generate Feedback

## How to Run
--- bash
  -- cd backend
  -- python app.py

--- Frontend
    -- Open index.html using Live Server
